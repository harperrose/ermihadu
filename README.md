# ermihadu
family memory website to allow one nuclear family to upload items and descriptions in the way they see fit

## Overview
This is an experimental personal project in which I am trying to use as a family information storage. This is a google-native, react-based content system. 

## Tech Stack
- Framework: React
- Language: TypeScript
- Styling: CSS
- CMS / Backend: Google Cloud, Google Drive, Google Sheet content management.
- Tooling: Google Cloud Platform, Google Sheets API, Google Drive API, Google Speech-to-Text API, Github Hosting, Spaceship Domain

## Features
- Core feature: Uploading items to be displayed in list
- Secondary feature: Speech to text integration for audio descriptions
- Nice-to-have feature: Family tree memebers links
